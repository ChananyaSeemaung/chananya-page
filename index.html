เกี่ยวกับฉัน

สวัสดีครับ ฉันชื่อ ชนันท์ญา สีม่วง เป็นนักพัฒนาเว็บที่ชื่นชอบการสร้างหน้าเว็บสวยงาม ใช้งานง่าย และเน้นการออกแบบที่ตอบโจทย์ผู้ใช้งาน พร้อมเรียนรู้เทคโนโลยีใหม่ๆ อย่างต่อเนื่อง

ทักษะ

  • HTML / CSS / JavaScript
  • Responsive Design
  • การพัฒนาเว็บพื้นฐาน
  • ออกแบบ UI / UX เบื้องต้น

ผลงานเด่น

  • Portfolio Page - ออกแบบและพัฒนาเว็บไซต์ส่วนตัวสำหรับแสดงผลงานและข้อมูลส่วนตัว
  • หน้าเว็บแบบ Responsive - สร้างเว็บไซต์ที่ดูดีทั้งบนคอมพิวเตอร์และมือถือ
  • โปรเจคการเรียนรู้เว็บ - ทดลองสร้างหน้าเว็บด้วย HTML/CSS, ปรับแต่งรูปภาพ และจัดวางเนื้อหาให้สวยงาม
